# GGSA promo page image assets

Logos and product photos for the `/ggsa` page are now **uploaded from the
admin UI** (Admin → Settings → GGSA Promo Page) and stored in Google Cloud
Storage. You normally don't need to put anything here.

This folder remains only as an optional fallback: if you'd rather serve a
static file, drop it here and paste its `/images/ggsa/<file>` path — but the
admin upload widgets are the supported path and will overwrite the stored URL.
